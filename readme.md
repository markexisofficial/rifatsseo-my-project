## 🎨 Design System

### Color Palette

```css
/* Primary Colors */
--primary-color: #3b82f6 (Blue)
--primary-hover: #2563eb
--accent-color: #8b5cf6 (Purple)

/* Dark Theme */
--bg-primary: #0a0e1a (Darkest background)
--bg-secondary: #111827 (Secondary background)
--bg-card: #1a1f2e (Card backgrounds)

/* Text Colors */
--text-primary: #f9fafb (Primary text - white)
--text-secondary: #d1d5db (Secondary text - light gray)
--text-muted: #9ca3af (Muted text)

/* Status Colors */
--success: #10b981 (Green)
--warning: #f59e0b (Orange)
--error: #ef4444 (Red)

/* Gradients */
--gradient-primary: linear-gradient(135deg, #3b82f6 0%, #8b5cf6 100%)
```

### Typography

**Primary Font (Body):** Inter  
- Weights: 300, 400, 500, 600, 700, 800
- Usage: All body text, paragraphs, descriptions

**Heading Font:** Poppins  
- Weights: 600, 700, 800
- Usage: All headings (H1-H6), numbers, statistics

**Font Sizes:**
- Base: 16px
- H1: 2.5rem - 4rem (responsive)
- H2: 2rem - 3rem (responsive)
- H3: 1.5rem - 2rem (responsive)
- Body: 1rem
- Small: 0.875rem

### Spacing System

- Container Max-Width: 1200px
- Section Padding: 80px vertical (60px on mobile)
- Card Padding: 2rem
- Grid Gaps: 2rem (1.5rem on mobile)

### Responsive Breakpoints

```css
Desktop: > 1024px (default)
Tablet: 768px - 1024px
Mobile: < 768px
Small Mobile: < 480px
```

## 🎨 Customization Guide

### Updating Contact Information

**Email Address:**
Update in all HTML files:
```html
<a href="mailto:rifatseoanalysis@gmail.com">rifatseoanalysis@gmail.com</a>
```

**Phone Number:**
Currently placeholder "(XXX) XXX-XXXX" - update in:
- About page bio section
- Contact page

**Social Media Links:**
Update `#` placeholders in footer:
```html
<a href="#" aria-label="LinkedIn">   <!-- Add LinkedIn URL -->
<a href="#" aria-label="WhatsApp">   <!-- Add WhatsApp link -->
```

### Changing Colors

Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #3b82f6;     /* Change primary blue */
    --accent-color: #8b5cf6;      /* Change accent purple */
    --success: #10b981;            /* Change success green */
}
```

### Adding Real Images

Replace placeholder sections with real images:

1. Create an `images/` directory
2. Add your images
3. Update placeholder divs:

```html
<!-- Replace this: -->
<div class="image-placeholder">
    <i class="fas fa-user-tie"></i>
</div>

<!-- With this: -->
<img src="images/profile.jpg" alt="Rakibur Rahman">
```

**Statistics:**
Update numbers in:
- Homepage hero section (trust badges)
- About page (career highlights)
- Portfolio page (stats)
- Testimonials page (trust stats)

**Case Studies:**
Customize with real client data:

### Connecting Contact Form

The form currently logs to console. To connect to a backend:

1. **Update `js/main.js`** form submission:
```javascript

```

2. **Alternative: Use FormSpree, Netlify Forms, or similar service**

### Current SEO Features

✅ **Semantic HTML5** structure

✅ **Alt text** placeholders for images

### Recommended Additions

1. **Add Meta Tags:**
```html
<meta property="og:title" content="Markexis - SEO Specialist">
<meta property="og:description" content="...">
<meta property="og:image" content="url-to-image">
<meta name="twitter:card" content="summary_large_image">
```

2. **Create sitemap.xml:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url><loc>https://markexis.com/</loc></url>
  <url><loc>https://markexis.com/about.html</loc></url>
  <!-- etc -->
</urlset>
```

3. **Create robots.txt:**
```
User-agent: *
Allow: /
Sitemap: https://markexis.com/sitemap.xml
```

4. **Add Schema Markup:**
- LocalBusiness schema
- Person schema
- Service schema
- Review schema

5. **Google Analytics & Search Console:**
- Add tracking code
- Verify ownership
- Submit sitemap
